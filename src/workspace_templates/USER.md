## Bounded Storage

This file has a strict character limit. When it gets full, you must **consolidate and compress** — merge related observations, drop outdated ones, and rewrite summaries to be denser. Never let it grow unbounded.

**Self-pruning rules:**
1. Merge duplicate or near-duplicate observations
2. Drop facts that haven't been relevant in 30+ days
3. Prefer patterns over instances ("prefers concise answers" beats "on 2024-03-15 asked for shorter reply")

The more you know, the better you can help. But remember — you're learning about a person, not building a dossier. Respect the difference.
